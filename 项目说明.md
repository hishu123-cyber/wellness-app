# 食术·中医体质养生 项目包

## 📦 内容说明

本压缩包包含食术养生 APP 的完整源码，可直接部署运行。

## 📁 目录结构

`
wellness_app/
├── frontend/              # 前端SPA
│   ├── app.html          # APP入口页面
│   ├── app.js            # APP主逻辑（124KB）
│   ├── styles.css        # 样式表（37KB）
│   ├── sw2.js            # Service Worker（缓存策略）
│   ├── manifest.json     # PWA清单
│   ├── index.html        # 官网首页
│   └── assets/           # 图标资源
├── server.js             # Express后端主入口
├── tea_service.js        # 茶养模块API
├── tcm_service.js        # 中医问诊API
├── nutritionist_service.js # 营养师API
├── ai_service.js         # AI体质辨识
├── pay.js                # 支付系统
├── seed-db.js            # 数据库初始化（含24张表+演示数据）
├── package.json          # 依赖配置
├── ecosystem.config.js   # PM2生产配置
├── .env                  # 环境变量（WELLNESS_SECRET）
└── scripts/             # 启动脚本
    ├── start_wellness.bat  # 一键启动服务器
    └── start_cf.bat       # 启动Cloudflare Tunnel
`

## 🚀 快速启动

### 开发环境
\\\ash
# 1. 安装依赖
npm install

# 2. 初始化数据库
node seed-db.js

# 3. 启动服务器
set WELLNESS_SECRET=wellness-secret-key
node server.js

# 4. 访问
# http://localhost:8000      → 官网
# http://localhost:8000/app → APP
\\\

### 生产环境（Windows）
双击运行 scripts/start_wellness.bat

## 📡 API列表

- /api/auth/* - 认证（登录/注册）
- /api/constitution/* - 体质测评
- /api/diary/* - 健康日记
- /api/recipes - 食疗药膳
- /api/solar-terms - 24节气
- /api/tea/* - 茶养模块
- /api/tcm/* - 中医问诊
- /api/nutritionist/* - 营养师
- /api/shop/* - 养生商城
- /api/points/* - 积分系统
- /api/vip/* - 会员系统
- /api/pay/* - 支付

## 📱 PWA

前端支持 PWA，可添加到手机桌面。
Service Worker (sw2.js) 策略：
- 导航请求（HTML）：networkFirst（始终获取最新）
- 静态资源（JS/CSS/图片）：cacheFirst + 版本戳

## 📊 数据库

24张表，包括：用户、体质测评、健康日记、食疗、节气、商城、订单、积分、VIP、中医问诊、营养师等。
