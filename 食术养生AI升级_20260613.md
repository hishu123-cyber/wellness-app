## ✅ 已完成模块

### 1. AI智能辨体引擎 (`ai_constitution_engine.js`)

**API路由：**
| 路由 | 方法 | 功能 |
|------|------|------|
| `/api/ai/tongue-analysis` | POST | AI舌象分析（舌色/舌体/舌苔/舌下络脉） |
| `/api/ai/voice-consultation` | POST | 语音问诊NLP分析 |
| `/api/ai/comprehensive-assessment` | POST | 综合辨体（问卷+舌象+语音融合） |
| `/api/ai/smart-shop` | GET | 一键配菜——体质适配商品排序 |
| `/api/ai/diet-plan` | GET | 个性化食疗方案生成 |
| `/api/ai/tongue-guide` | GET | 舌象采集指导 |
| `/api/ai/voice-questions` | GET | 语音问诊引导问题 |

**核心能力：**
- 舌象知识库：6种舌色 × 5种舌体 × 6种舌苔 × 4种舌下络脉
- 语音问诊：8种体质关键词映射，自动提取证候
- 综合辨体：问卷50% + 舌象30% + 语音20% 加权融合
- 一键配菜：24种食材 × 9种体质适配度评分
- 食疗方案：分人群（老年/中年/女性/婴幼儿/青年）场景化推荐

### 2. 用户画像与智能推荐 (`user_profile_engine.js`)

**API路由：**
| 路由 | 方法 | 功能 |
|------|------|------|
| `/api/user-profile/tags` | GET | 获取用户标签体系 |
| `/api/user-profile/home` | GET | 千人千面首页数据 |
| `/api/user-profile/archive` | GET | 健康档案聚合 |
| `/api/user-profile/push` | GET | 差异化推送内容 |

**核心能力：**
- 五维标签：体质标签 + 症状标签 + 行为标签 + 生命周期标签 + 偏好标签
- 五大人群分层：老年人/中年人/女性专属/婴幼儿/青年人
- 千人千面首页：根据人群定制快捷入口、推送内容、商品推荐
- 健康档案：体质历程 + 问诊记录 + 购买偏好 + 健康趋势

### 3. 节气日历+任务打卡 (`seasonal_calendar_service.js`)

**API路由：**
| 路由 | 方法 | 功能 |
|------|------|------|
| `/api/seasonal/current-wellness` | GET | 当前节气养生方案 |
| `/api/seasonal/daily-tasks` | GET | 今日个性化任务 |
| `/api/seasonal/checkin` | POST | 完成任务打卡 |
| `/api/seasonal/checkin-history` | GET | 打卡历史 |
| `/api/seasonal/calendar` | GET | 24节气养生日历 |
| `/api/seasonal/redeem-coupon` | POST | 积分兑换优惠券 |

**核心能力：**
- 24节气完整养生方案：每个节气含主题、饮食重点、任务、宜吃忌口、体质专属方案
- 每日任务生成：节气专属(3) + 通用(3) + 人群专属(1)
- 连续打卡机制：3天×1.1 / 7天×1.3 / 14天×1.5 / 21天×1.8 / 30天×2.0 积分加成
- 积分兑换：100分→5元券 / 200分→10元券 / 150分→免邮券
