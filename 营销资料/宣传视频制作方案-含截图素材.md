## 四、剪辑制作指南

### 工具推荐

| 工具 | 用途 |
|------|------|
| **剪映PC版** | 主剪辑工具（免费功能足够） |
| **Canva** | 制作封面图、文字叠加、大字幕 |
| **手机录屏** | 获取APP页面动效素材（如有需要） |

### 操作步骤

```
Step 1: 打开APP → 截取所有页面截图（已保存）
Step 2: 导入剪映 → 按分镜顺序排列截图
Step 3: 每张截图设置"缩放"动画（从100%→105%，模拟推进感）
Step 4: 截图之间用"滑动"转场（0.3秒）
Step 5: 录制旁白 → 对齐时间线
Step 6: 添加字幕 → 白色大号字+半透明黑底
Step 7: 配BGM → 搜索"国风""养生""温暖治愈"
Step 8: 最后添加"私信了解详情"行动号召

预计制作时间：2-3小时
```

### 动效参数建议

| 动效 | 参数 |
|------|------|
| 截图入场 | 放大（120%→100%）+ 渐入 0.5秒 |
| 页面切换 | 左滑（0.2-0.3秒） |
| 文字出现 | 打字机效果（速度适中） |
| 图表动画 | 线性绘制（2秒，配合旁白） |
| 情绪转折点（第8镜） | 音乐高潮+画面闪白转场 |

### 字体推荐

| 用途 | 字体 |
|------|------|
| 旁白字幕 | 思源黑体 / 微软雅黑 Bold |
| 标题大字 | 思源宋体 / 站酷高端黑 |
| 封面文字 | 书法体（如：叶根友毛笔行书） |

### 多平台适配裁剪

| 版本 | 时长 | 用途 | 裁剪说明 |
|------|------|------|---------|
| 完整版 | 65秒 | 视频号/抖音 | 全本完整 |
