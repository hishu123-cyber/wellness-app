## 四、制作指南

### 工具推荐
| 工具 | 用途 |
|------|------|
| 剪映APP | 剪辑、加字幕、配乐、特效（手机端免费即可） |
| Canva | 制作封面图和文字排版镜头 |
| 系统录屏 | iOS自带 / 安卓长截屏录屏功能 |
| Pexels / Pixabay | 搜 "elderly health" "Chinese medicine" 补充素材 |

### 剪辑注意
- 手机录屏建议 **2倍速后慢放**，看起来更流畅
- 字幕用 **白色大号字 + 半透明黑色底**，照顾中老年受众
- BGM 搜索关键词：「温暖」「治愈」「轻快」「国风养生」
- 视频封面：绿色底色 + 白色大字「1分钟了解你的体质」

### 多版本适配

| 平台 | 时长 | 建议版本 |
|------|------|----------|
| 微信朋友圈 | 15-30秒 | 精简版：仅保留第1+3+4+10镜 |
| 视频号 | 60秒 | 完整版如上 |
| 抖音 | 45-60秒 | 完整版，前3秒加爆点 |
| 微信群发 | 15秒 | 仅展示+行动号召 |

### 精简版（15秒，朋友圈专用）
```
镜1(0-3s): 体检报告愁眉不展
镜3(3-7s): 体质测评快速展示
镜4(7-11s): 食疗推荐
镜10(11-15s): "私信了解详情"
```
